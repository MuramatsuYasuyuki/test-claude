# test-claude

## このリポジトリについて

このリポジトリは、Claude Code の動作確認や各種機能を試すためのテスト用リポジトリです。

コードの編集・コミット・プッシュといった一連の開発フローや、GitHub 連携、自動化ワークフローなどを実際に動かして検証することを目的としています。本番運用を想定したものではなく、検証・学習・実験のためのサンドボックスとして利用します。

## 使い方

### 1. リポジトリを取得する

```bash
git clone https://github.com/MuramatsuYasuyuki/test-claude.git
cd test-claude
```

### 2. 作業用ブランチを作成する

`main` ブランチで直接作業せず、目的ごとにブランチを切って作業します。

```bash
git checkout -b feature/my-experiment
```

### 3. 変更してコミットする

ファイルを編集したら、変更内容をコミットします。

```bash
git add .
git commit -m "変更内容を説明するメッセージ"
```

### 4. プッシュする

作業ブランチをリモートにプッシュします。

```bash
git push -u origin feature/my-experiment
```

### 5. プルリクエストを作成する（任意）

GitHub 上でプルリクエストを作成し、変更内容をレビュー・マージできます。

## ディレクトリ構成

検証目的に応じて自由にファイルやディレクトリを追加してください。不要になったものは削除して問題ありません。

## 補足

- このリポジトリは実験用のため、内容は予告なく変更・削除されることがあります。
- 機密情報（API キーやパスワードなど）はコミットしないでください。`.gitignore` で `.env` などを除外しています。
